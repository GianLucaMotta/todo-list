# Instruções
	1 - A pasta lararestful é responsável pela api laravel que fornece os dados, dentro dela encontrará um 		arquivo README.md com instruções para a criação do banco de dados e funcionamento da API Laravel.
	
	2 - A pasta vue é responsável pelo front-end da aplicação, que consome os dados da API Laravel, dentro 	dela encontrará um arquivo README.md com instruções para execução do front-end.

A API Laravel e o front em Vue.js são executados separadamente e unicamente, mas um necessita do outro.
